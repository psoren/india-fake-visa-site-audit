# Indian Gov Tech Audit

Section-by-section technical teardowns of Indian government websites — what's actually shipped to the browser, what's broken, and what would help most. Also tracks look-alike / fake e-visa sites and how to spot them in the markup.

**Live site:** https://psoren.github.io/india-fake-visa-site-audit/

Audits live under `src/content/docs/audits/`. Scam-tracker pages under `src/content/docs/scams/`. The site is built with Astro + Starlight and deploys to GitHub Pages via the workflow in `.github/workflows/deploy.yml`.
