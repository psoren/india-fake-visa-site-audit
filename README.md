# Indian Gov Tech Audit

Section-by-section technical teardowns of Indian government websites — what's actually shipped to the browser, what's broken, and what would help most.

**Live site:** https://psoren.github.io/indian-gov-tech-audit/

Audits live under `src/content/docs/audits/`. The site is built with Astro + Starlight and deploys to GitHub Pages via the workflow in `.github/workflows/deploy.yml`.
